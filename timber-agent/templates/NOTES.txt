To verify that the Timber Agent has started, run:

kubectl get pods \
  --namespace={{ .Release.Namespace }} \
  -l "app={{ template "timber.name" . }},release={{ .Release.Name }}"

To view the Timber Agent logs, run:

kubectl logs \
  --namespace={{ .Release.Namespace }} \
  -l "app={{ template "timber.name" . }},release={{ .Release.Name }}" \
  -c {{ template "timber.name" . }}
